**Yet another arcanoid game**

Requirements:
- https://nodejs.org/en/
- https://www.npmjs.com/

Installation:
- install dependencies using `npm install`
- run web server `node node_modules/http-server/bin/http-server -p 8080
`
- visit: `http://localhost:8080/index.html`
